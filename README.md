## Sobre o Projeto                                          

Este projeto foi desenvolvido para a Global Solution 2026 o objetivo do projeto é monitorar uma cápsula espacial através de sensores de temperatura, luminosidade e vibração. Os dados são processados pelo Arduino e exibidos em um display LCD.                                         

Além disso, o sistema possui LEDs e buzzer para indicar diferentes níveis de alerta.                                    

## Componentes Utilizados                             

- Arduino Uno                         
- Protoboard                             
- Sensor TMP36                          
- Fotorresistor                          
- Potenciômetro                         
- Display LCD 16x2                        
- LED verde                     
- LED amarelo                        
- LED vermelho                    
- Buzzer                       
- Resistores                     
- Jumpers                                   

## Funcionamento                                                         

O sistema realiza a leitura dos sensores e verifica se os valores estão dentro dos limites definidos.                                                    

Existem três estados de funcionamento:                 

- Normal: LED verde aceso.                                 
- Atenção: LED amarelo aceso.                                    
- Crítico: LED vermelho aceso e buzzer ativado.                                                        

Os valores dos sensores são mostrados em tempo real no display LCD.                                                                

## Funcionalidade Adicional                                                     
                       
Foi adicionado um nível intermediário de alerta utilizando o LED amarelo, permitindo identificar situações de atenção antes de chegar ao estado crítico.
